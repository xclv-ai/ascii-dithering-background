ASCII Dithering Background

Experience the mesmerizing beauty of ASCII art as your website background

🚀 One-Line Integration

Add this ASCII dithering background to any website with just one line of code:

<script src="https://cdn.jsdelivr.net/gh/xclv-ai/ascii-dithering-background@main/ascii-background.js"></script>

That's it! The background automatically appears behind your content without affecting your existing layout.

✨ Features

  • Zero Dependencies: Pure vanilla JavaScript, no frameworks needed
  • Lightweight: Less than 5KB minified
  • Responsive: Automatically adapts to any screen size
  • Non-intrusive: Won't break your existing CSS or JavaScript
  • Customizable: Easy to modify colors, patterns, and opacity
  • Performance Optimized: Uses canvas for smooth rendering

🎮 Control Functions

Programmatically control the background after it loads:

// Change opacity (0 = invisible, 1 = fully visible)
ASCIIDithering.setOpacity(0.5);

// Change color
ASCIIDithering.setColor('#ff0000');

// Toggle visibility
ASCIIDithering.toggle();

// Remove completely
ASCIIDithering.remove();
